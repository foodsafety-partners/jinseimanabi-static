【徹底解説】ChromeでのMetaMask（仮想通貨ウォレット）のインストール方法（写真で分かりやすく解説）

仮想通貨

MetaMaskのインストール方法（１０分程度）

MetaMaskは仮想通貨用の財布です。

仮想通貨を始めるときの難所（英語が面倒なだけ）ですが、

これさえ完了すれば、仮想通貨デビューにグンと近づけます。

英語が出てきますが、１０分くらいなので一緒に乗りきりましょう。

まずは最初に注意事項があるので、全体の概要を説明していきます。

（ガンガン進めてしまって、「失敗した！」とならないように）

MetaMaskを下記のサイトからインストールしてください。

偽物のサイトがあるようですので、リンク貼ってます。アドレスがmetamaskとなっているか、しっかり確認して下さい。metamosk（メタモスク）とかいう偽サイトありそうです（笑）

https://metamask.io/

ちなみに、MetaMaskという単語をツイートすると、詐欺リプが付いてきますので、引っかからないように注意してください。

基本的に流れに沿ってインストールすれば大丈夫ですが、

『⑥シードフレーズのバックアップをスクショ、紙などで記録』

だけは注意してください。

これを忘れると、パスワードを忘れた時などにすべてを失います。

もちろん、人に教えてもダメです。

シードフレーズという聞きなれない言葉ですが、パスワードと同じものと思ってください。

では、下記の流れでサクッとMetaMaskをインストールしていきましょう。

①「Download now」をクリック。

 

②「Install MetaMask for Chrome」をクリック。

わたしはChromeでインストールしました。操作性が良いのでPCでのインストールをおすすめします。

 

③Chromeに追加をクリック

④「ウォレットの作成」をクリック

⑤パスワードの作成

⑥シードフレーズのバックアップをスクショ、紙などで記録

⑦完了

これでMetaMaskが入りました。

⑧確認

Chromeを立ち上げると右上にキツネが現れます。そこをクリックするとログイン画面が出てきます。

おめでとうございます！

あとは仮想通貨を購入して、MetaMaskに入金すれば、イーサリアムでNFT買ったり、ビットコインでDeFi運用したりできます。
